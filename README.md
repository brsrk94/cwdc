# CWDC Directory

A premium resource directory for Cybersecurity, Web Development, Data Science, and Competitive Coding.

## Features

- **Modern UI**: Dark theme with neon accents and spotlight hover effects.
- **Responsive Design**: Works seamlessly on desktop and mobile.
- **Comprehensive Sections**:
  - **Cybersecurity**: Red/Blue/Purple teams, CTF resources (Rev Eng, Pwn, Crypto, etc.).
  - **Web Development**: Frontend, Backend, DevOps.
  - **Data Science**: ML, DL, AI, Math.
  - **Competitive Coding**: Algorithms, Data Structures, Platforms.

## Tech Stack

- React
- Vite
- Vanilla CSS (Custom Variables & Animations)
- React Router DOM

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
